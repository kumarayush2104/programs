1. Introduction to HTML

HTML stands for HyperText Markup language. It is used to create structures and content of the webpage which will be displayed in the browser.

2. Why do we learn HTML ?

We learn HTML to create structures and content of multiple webpages. Those multiple webpages can be combined into a website.

3. What is HTTP ?

HTTP stands for HyperText Transfer Protocol, which is used to create a communication between server and client for transferring data and information between computers.

4. What are the basic elements / tags of HTML ?

Some of the basic HTML tags are:
1. <h1> to <h6>: header tags
2. <p>: paragraph tag
3. <a>: anchor tag
4. <br>: break tag
5. <hr>: horizontal line tag
6. <b>: bold tag
7. <u>: underline tag

5. What are the advantages and disadvantages of HTML ?

Advantages HTML are:
1. No need of compilers/interpreters
2. Easy to learn and write
3. Can embed medias like images, audio and video
4. Loose syntax
5. Every browser supports HTML.

Disadvantages HTML are:
1. Insufficient for Dynamic Pages
2. Needs extra languages like CSS and JS for themeing and styling
3. Sometimes finding the error may be difficult
4. Unpredicatable behaviour across the browser
